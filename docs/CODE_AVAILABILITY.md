# Code availability

The GenomeCF software release includes:

- package source in `genomecf/`
- publication generators in `src/`
- scripts in `scripts/`
- tests in `tests/`
- documentation in `docs/`

Primary entry points:

```bash
genomecf evaluate ...
genomecf external ...
genomecf synth ...
genomecf build-paper
genomecf build-website
```
