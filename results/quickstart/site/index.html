GenomeCF

GenomeCF is a counterfactual validation standard for DNA sequence models. It combines core regulatory tasks, external biological assays, matched-negative evaluation, GC-bin robustness, foundation-model comparison, and a reusable synthetic mechanism suite.

This local site is generated from the release registry and publication tables.

Release highlights

  • Core official benchmark with 6-mer, CNN, RC-aug CNN, DNABERT-2, and Caduceus-Ph.
  • External biological validation across TF-binding, histone-mark, and variant-effect assay families.
  • GenomeCF-Synth shortcut-conflict tasks for controlled rule-vs-shortcut testing.
  • Reporting checklist, canonical registry, and locally reproducible paper build.
