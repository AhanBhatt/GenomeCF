# Data availability

GenomeCF uses a mix of:

- lightweight public task bundles in `data/`
- larger local benchmark assets in `../local_runtime_assets/data/` when present
- exported manifests in `configs/`
- external assay tasks integrated through local loaders and manifests

Key files:

- `configs/task_manifests.jsonl`
- `results/release/benchmark_registry.csv`
- `results/release/external_validation_summary.csv`

Upstream datasets retain their original licenses and usage terms.
