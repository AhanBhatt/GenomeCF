# Paper Artifacts

This directory is the canonical home for the GenomeCF manuscript.

Main files:

- `genomecf_report.tex`
- `genomecf_report.pdf`
- `refs.bib`
- `neurips_2019.sty`

Supporting layout:

- `build/`
  - transient LaTeX build products
- `archive/`
  - older class-project proposal/report artifacts kept for reference
- `template/`
  - original template material kept for reference

`genomecf build-paper` compiles the manuscript directly from this directory and writes the final PDF here.
